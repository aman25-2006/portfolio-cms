# Portfolio CMS

A full-stack personal portfolio built with Next.js, TypeScript, Tailwind CSS, Supabase, and Vercel.

## 🚀 Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- Vercel

## ✨ Features

- Responsive Portfolio
- Secure Admin Dashboard
- Authentication
- CRUD Operations
- Resume Upload
- Contact Form
- SEO Optimized
- Dark/Light Mode

## 📦 Installation

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the application.
